# AI Gallery.

##### Check out original website
<https://openai.com/dall-e-2/>

It is an AI-driven image generator app based on the DALLE-E bot from OPENAI

# Prerequisities

- React
- OpenAi
- TailwindCSS
- JavaScript
- MongoDB
- Cloudinary
- NodeJS
- Express
